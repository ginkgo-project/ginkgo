Detailed Description

This is a limited implementation of the CUDA grid_group that works even on devices that do not support device-wide synchronization and without special kernel launch syntax.

Note that this implementation (as well as the one from CUDA's cooperative groups) does not support large grids, since it uses 32 bits to represent sizes and ranks, while at least 73 bits (63 bit grid + 10 bit block) would have to be used to represent the full space of thread ranks.
