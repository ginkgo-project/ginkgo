Detailed Description

template<typename ValueType = default_precision, typename IndexType = int32>
class gko::preconditioner::BlockJacobiFactory< ValueType, IndexType >

This factory is used to create a block-Jacobi preconditioner from the operator A, by inverting the diagonal blocks of the operator.

The factory generates a BlockJacobi object witch stores the preconditioner.

Template Parameters
ValueTypeprecision of matrix elements
IndexTypeintegral type used to store pointers to the start of each block
