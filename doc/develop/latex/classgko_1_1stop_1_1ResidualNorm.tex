\subsubsection*{template$<$typename Value\+Type = default\+\_\+precision$>$\newline
class gko\+::stop\+::\+Residual\+Norm$<$ Value\+Type $>$}

The \mbox{\hyperlink{classgko_1_1stop_1_1ResidualNorm}{Residual\+Norm}} class is a stopping criterion which stops the iteration process when the actual residual norm is below a certain threshold relative to. 


\begin{DoxyEnumerate}
\item the norm of the right-\/hand side, norm(residual) / norm(right\+\_\+hand\+\_\+side) $<$ threshold
\item the initial residual, norm(residual) / norm(initial\+\_\+residual) $<$ threshold.
\item one, norm(residual) $<$ threshold.
\end{DoxyEnumerate}

For better performance, the checks are run on the executor where the algorithm is executed.

\begin{DoxyNote}{Note}
To use this stopping criterion there are some dependencies. The constructor depends on either {\ttfamily b} or the {\ttfamily initial\+\_\+residual} in order to compute their norms. If this is not correctly provided, an exception \+::gko\+::\+Not\+Supported() is thrown. 
\end{DoxyNote}
