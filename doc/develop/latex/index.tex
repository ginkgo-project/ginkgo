Documentation on aspects such as the build system, can be found at the \mbox{\hyperlink{install_ginkgo}{Installation Instructions}} page. The \mbox{\hyperlink{Examples}{Example programs}} can help you get started with using Ginkgo.\hypertarget{index_autotoc_md19}{}\doxysubsubsection{Modules}\label{index_autotoc_md19}
The Ginkgo library can be grouped into \href{modules.html}{\texttt{ modules}} and these modules form the basic building blocks of Ginkgo. The modules can be summarized as follows\+:


\begin{DoxyItemize}
\item \mbox{\hyperlink{group__Executor}{Executors}} \+: Where do you want your code to be executed ?
\item \mbox{\hyperlink{group__LinOp}{Linear Operators}} \+: What kind of operation do you want Ginkgo to perform ?
\begin{DoxyItemize}
\item \mbox{\hyperlink{group__solvers}{Solvers}} \+: Solve a linear system for a given matrix.
\item \mbox{\hyperlink{group__precond}{Preconditioners}} \+: Precondition a system for a solve.
\item \mbox{\hyperlink{group__mat__formats}{Sp\+MV employing different Matrix formats}} \+: Perform a sparse matrix vector multiplication with a particular matrix format.
\end{DoxyItemize}
\item \mbox{\hyperlink{group__log}{Logging}} \+: Monitor your code execution.
\item \mbox{\hyperlink{group__stop}{Stopping criteria}} \+: Manage your iteration stopping criteria. 
\end{DoxyItemize}
