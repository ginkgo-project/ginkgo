\doxysubsection{Detailed Description}
This class represents two levels in a multigrid hierarchy. 

The \mbox{\hyperlink{classgko_1_1multigrid_1_1MultigridLevel}{Multigrid\+Level}} is an interface that allows to get the individual components of multigrid level. Each implementation of a multigrid level should inherit from this interface. Use Enable\+Multigrid\+Level$<$\+Value\+Type$>$ to implement this interface with composition by default.
