\subsubsection*{template$<$typename Value\+Type = gko\+::default\+\_\+precision, typename Index\+Type = gko\+::int32$>$\newline
class gko\+::factorization\+::\+Ilu$<$ Value\+Type, Index\+Type $>$}

Represents an incomplete LU factorization -- I\+L\+U(0) -- of a sparse matrix. 

More specifically, it consists of a lower unitriangular factor $L$ and an upper triangular factor $U$ with sparsity pattern $\mathcal S(L + U)$ = $\mathcal S(A)$ fulfilling $LU = A$ at every non-\/zero location of $A$.
