Detailed Description

template<typename ValueType = default_precision, typename IndexType = int32>
class gko::preconditioner::AdaptiveBlockJacobi< ValueType, IndexType >

A block-Jacobi preconditioner is a block-diagonal linear operator, obtained by inverting the diagonal blocks of another operator.

This is a variant of the BlockJacobi preconditioner which can use lower precision to store well-conditioned diagonal blocks, and thus improve the performance of preconditioner application by reducing the amount of memory that has to be read to apply the precondidionter.

However, there is a trade-off in terms of longer preconditioner generation due to extra work required to compute the condition numbers. This step is necessary to preserve the regularity of the diagonal blocks.

Template Parameters
ValueTypehighest precision of matrix elements, the precision of vectors used for the apply() method and the precision in which all computation is performed
IndexTypeintegral type used to store pointers to the start of each block
